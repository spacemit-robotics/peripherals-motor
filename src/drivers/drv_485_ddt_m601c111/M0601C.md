# M0601C_111 电机通信与控制说明

## 1. 通信协议
- **通信方式**：RS485 总线协议
- **波特率**：115200
- **数据位**：8bit
- **停止位**：1bit
- **奇偶校验位**：无
- **数据长度**：10 字节
- **回复形式**：一问一答
- **最高速率**：500Hz

## 2. 数值转换方式
- **电流环模式**：给定值范围 `-32767 ~ 32767`，对应量程 `-8 ~ 8A`（非电机电流范围），有符号16位。
- **速度环模式**：给定值范围 `-330 ~ 330`，单位 `rpm`，有符号16位。
- **位置环模式**：给定值范围 `0 ~ 32767`，对应 `0° ~ 360°`，无符号16位。

## 3. 电机使能、失能方式（帧格式）
### 使能
通过切换电机模式实现使能，默认模式速度环
发送格式：
- `DATA[0]`: ID
- `DATA[1]`: 0xA0
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: 模式值 (`0x01` 电流环, `0x02` 速度环, `0x03` 位置环)
### 失能
- **失能释放**：当 SDK 下发 IDLE 时，驱动底层会主动将电机**切换电流环模式，并下发 0A 电流**。此时电机将完全失去电磁力矩，处于**无阻尼的自由滑行（释放）状态**，可以通过外力轻松转动。
  - **重要提醒**：此失能方式并不是物理上的断电失能，驱动器内部控制环路仍在正常运行，该设计仅仅是为了使电机快速进入顺从状态，因此**不适合作为长时间待机状态使用**。

## 4. 电机参数的读取和下发方式（帧格式）
### 4.1 驱动电机转动（下发）
- `DATA[0]`: ID
- `DATA[1]`: 0x64
- `DATA[2], DATA[3]`: 速度/电流/位置给定 高8位/低8位
- `DATA[4], DATA[5]`: 0
- `DATA[6]`: 加速时间
- `DATA[7]`: 刹车（0xFF为刹车，速度环模式有效）
- `DATA[8]`: 0
- `DATA[9]`: CRC8

### 4.2 电机驱动反馈（读取）
- `DATA[0]`: ID
- `DATA[1]`: 模式值
- `DATA[2], DATA[3]`: 转矩电流 高8位/低8位
- `DATA[4], DATA[5]`: 速度 高8位/低8位
- `DATA[6], DATA[7]`: 位置 高8位/低8位
- `DATA[8]`: 故障码
- `DATA[9]`: CRC8

### 4.3 获取其他反馈（下发查询）
- `DATA[0]`: ID
- `DATA[1]`: 0x74
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

### 4.4 其他反馈回复（读取）
- `DATA[0]`: ID
- `DATA[1]`: 模式值
- `DATA[2], DATA[3]`: 转矩电流 高8位/低8位
- `DATA[4], DATA[5]`: 速度 高8位/低8位
- `DATA[6]`: 绕组温度
- `DATA[7]`: U8位置值（0~255对应0~360°）
- `DATA[8]`: 故障码
- `DATA[9]`: CRC8

## 5. 数据阈值说明
- **母线过流保护阈值**：3A（过流发生后触发停机保护，5S后解除）。
- **电机过温保护阈值**：80℃（温度低于阈值5℃后解除保护）。
- **相电流保护阈值**：4.6A（过流发生后触发停机保护，5S后解除）。
- **堵转保护**：堵转持续时间超过5S触发保护，5S后解除。

## 6. 其他控制参数与特殊指令
### 6.1 加速时间设定
在驱动电机转动指令中，通过 `DATA[6]` 设定电机的加速时间。
- **说明**：写 `0` 加速最快（默认加速时间），数值越大加速越慢。
- **示例**：`01 64 00 1E 00 00 00 00 00 18`（设定速度 30RPM，加速时间默认为 0）。
- **示例**：`01 64 00 1E 00 00 0A 00 00 72`（设定速度 30RPM，加速时间为 10）。

### 6.2 刹车控制
仅在**速度环模式**下有效，在驱动电机转动指令中通过 `DATA[7]` 控制。
- **`0xFF`**：执行刹车。
- **`0x00`**：不刹车。
- **示例**：`01 64 00 00 00 00 00 FF 00 D1`（电机 1 刹车）。

### 6.3 电机 ID 设置
- **前提**：设置 ID 时请保证总线上只有一个电机，每次上电只允许设置一次。
- **条件**：电机接收到 5 次 ID 设置指令后进行设置生效。
- **格式**：`AA 55 53 [ID] 00 00 00 00 00 00`。其中 `DATA[3]` 为需要设置的目标电机 ID。
- **示例**：`AA 55 53 01 00 00 00 00 00 00`（设置电机 ID 为 1，需连续发送 5 次）。

### 6.4 电机 ID 查询
- **格式**：`DATA[0]` 固定写 `0xC8`。
- **指令**：`C8 64 00 00 00 00 00 00 00 DE`。
