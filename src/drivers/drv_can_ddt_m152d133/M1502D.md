# M1502D_133 电机通信与控制说明

## 1. 通信协议
- **通信方式**：CAN 总线
- **速率**：1000Kbps
- **帧类型**：标准帧
- **数据长度**：8 字节
- **反馈数据方式**：主动上报及查询方式两种

## 2. 数值转换方式
- **开环模式**：给定值 `0 ~ -32767` 对应反转零速到最高速，`0 ~ 32767` 对应正转零速到最高速。
- **电流环模式**：给定值范围 `-16383 ~ 16383`，对应 `-55A ~ 55A`。
- **速度环模式**：给定值范围 `-21000 ~ 21000`，对应转速 `-210rpm ~ 210rpm`。
- **位置环模式**：给定值范围 `0 ~ 32767`，对应 `0° ~ 360°`。

## 3. 电机使能、失能方式（帧格式）
通过模式设置指令（标识符 `0x105`）进行使能/失能，且能够分别指定电机 ID 1~8。
发送格式：
- **标识符**：`0x105`
- `DATA[0]`: ID 1 模式值
- `DATA[1]`: ID 2 模式值
...
- `DATA[7]`: ID 8 模式值

**模式值说明**：
- `0x09`：失能电机
- `0x0A`：使能电机（默认使能）
*(其它值如 `0x00`电压开环, `0x01`电流环, `0x02`速度环, `0x03`位置环 用于切换控制模式)*

## 4. 电机参数的读取和下发方式（帧格式）
### 4.1 驱动命令给定（下发）
控制报文标识符分为两组，分别控制 ID 1~4 和 ID 5~8 的电机：
- **标识符**：`0x32` （对应电机 ID 1~4）
  - `DATA[0], DATA[1]`: 电机1给定 高8位/低8位
  - `DATA[2], DATA[3]`: 电机2给定 高8位/低8位
  - `DATA[4], DATA[5]`: 电机3给定 高8位/低8位
  - `DATA[6], DATA[7]`: 电机4给定 高8位/低8位
- **标识符**：`0x33` （对应电机 ID 5~8，数据域类似对应电机5~8）

### 4.2 接收端：反馈相关值（读取）
- **标识符**：`0x96 + 电机ID` (即 0x97 ~ 0x9E)
- `DATA[0], DATA[1]`: 速度测量值 高8位/低8位
- `DATA[2], DATA[3]`: 转矩电流测量值 高8位/低8位
- `DATA[4], DATA[5]`: 位置测量值 高8位/低8位
- `DATA[6]`: 故障值
- `DATA[7]`: 当前模式

**反馈值范围说明**：
- **速度值范围**：`-21000 ~ 21000`，对应 `-210RPM ~ 210RPM`
- **转矩电流值**：`-32767 ~ 32767`，对应 `-55A ~ 55A`
- **位置值范围**：`0 ~ 65535` (可通过机械零位标定将反馈值设置为 `0 ~ 32767`)，对应 `0 ~ 360°`

### 4.3 查询操作（查询方式下下发）
- **标识符**：`0x107`
- `DATA[0]`: 查询目标内容（例如：0x01速度, 0x02电流, 0x04位置等）
- `DATA[1]`: 电机 ID
- `DATA[2]`: 自定义值（区分返回帧）
- `DATA[3] ~ DATA[7]`: 保留

## 5. 数据阈值说明
- **欠压 1**：`18V < 母线电压 < 22V`
- **欠压 2**：`母线电压 < 18V`（停机）
- **过压**：`母线电压 > 63V`（停机）
- **过流**：`母线电流 > 20A`（停机）
- **过速**：`转速 > 300RPM`（停机）
- **过温 1**：`电机绕组温度 > 80℃`
- **过温 2**：`电机绕组温度 > 120℃`（停机）
- **堵转**：默认电流 `> 5A` 并且转速为 `0`（停机）
