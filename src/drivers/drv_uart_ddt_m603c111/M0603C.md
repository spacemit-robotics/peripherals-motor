# M0603C_111 电机通信与控制说明

## 1. 通信协议
- **通信方式**：UART 协议控制
- **波特率**：115200
- **数据位**：8bit
- **停止位**：1bit
- **奇偶校验位**：无
- **数据长度**：10 字节
- **回复形式**：一问一答
- **最高速率**：500Hz

## 2. 数值转换方式
- **开环（电流环）模式**：给定值范围 `-32767 ~ 32767`，对应量程 `-8A ~ 8A`。
- **速度环模式**：给定值范围 `-2100 ~ 2100`，对应 `-210rpm ~ 210rpm`，有符号16位。
- **位置环模式**：给定值范围 `0 ~ 32767`，对应 `0° ~ 360°`，无符号16位。

## 3. 电机使能、失能方式（帧格式）
### 上电即使能
模式切换发送格式：
- `DATA[0]`: ID
- `DATA[1]`: 0xA0
- `DATA[2]`: 模式值，模式值 (`0x00` 开环, `0x02` 速度环, `0x03` 位置环)
- `DATA[3] ~ DATA[8]`: 0
- `DATA[9]`: CRC8
    * 切换位置环电机速度需低于 10rpm
反馈：
- `DATA[0]`: ID
- `DATA[1]`: 0xA0
- `DATA[2]`: 模式值，模式值 (`0x00` 开环, `0x02` 速度环, `0x03` 位置环)
- `DATA[3] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

### 软失能（仅制动停转，力矩保持，原地锁死）：
发送格式：
- `DATA[0]`: ID
- `DATA[1]`: 0x64
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: CRC8
    

## 4. 电机参数的读取和下发方式（帧格式）
### 4.1 驱动电机转动（下发）
- `DATA[0]`: ID
- `DATA[1]`: 0x64
- `DATA[2], DATA[3]`: 速度/位置给定 高8位/低8位
- `DATA[4], DATA[5]`: 0
- `DATA[6]`: 加速时间（默认写 0 加速最快，可设置如 1、10 等，速度环模式有效，每 1rpm 的加速时间，单位为 1ms）
- `DATA[7]`: 刹车（0xFF为刹车，其他不刹车 速度环模式有效）
- `DATA[8]`: 0
- `DATA[9]`: CRC8

### 4.2 电机驱动反馈（读取）
- `DATA[0]`: ID
- `DATA[1]`: 0x64
- `DATA[2], DATA[3]`: 速度 高8位/低8位
- `DATA[4], DATA[5]`: 电流 高8位/低8位
- `DATA[6]`: 加速时间
- `DATA[7]`: 温度
- `DATA[8]`: 故障码
- `DATA[9]`: CRC8

### 4.3 获取其他反馈（下发查询）
- `DATA[0]`: ID
- `DATA[1]`: 0x74
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

### 4.4 其他反馈回复（读取）
- `DATA[0]`: ID
- `DATA[1]`: 0x74
- `DATA[2] ~ DATA[5]`: 里程圈数 高8位到低8位（32位）
- `DATA[6], DATA[7]`: 位置 高8位/低8位
- `DATA[8]`: 故障码
- `DATA[9]`: CRC8

    里程圈数：计圈范围- 2,147,483,647 到 2,147,483,647，重新上电会清 0
    位置值：0~32767 对应 0~360°

## 5. 电机 ID 查询与设置指令
### 5.1 电机 ID 设置指令（下发）
设置 ID 时请保证总线上只有一个电机，每次上电只允许设置一次，电机接收到 5 次 ID 设置指令后生效。
- `DATA[0]`: 0xAA
- `DATA[1]`: 0x55
- `DATA[2]`: 0x53
- `DATA[3]`: 目标 ID（如 0x01, 0x02等）
- `DATA[4] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

### 5.2 电机 ID 查询指令（下发查询）
- `DATA[0]`: 0xC8
- `DATA[1]`: 0x64
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: 0xDE (CRC校验位)

## 6. 数据阈值说明
- **母线过流保护阈值**：2.8A（过流发生后触发停机保护，5S后解除）。
- **电机过温保护阈值**：80℃（温度低于阈值5℃后解除保护）。
- **堵转保护**：堵转持续时间超过5S触发保护，5S后解除。

## 7. 获取模式反馈
### 7.1 模式查询指令（下发查询）
- `DATA[0]`: ID
- `DATA[1]`: 0x75
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

### 7.2 模式反馈回复（读取）
- `DATA[0]`: ID
- `DATA[1]`: 0x75
- `DATA[2]`: 模式值 (`0x00` 开环, `0x02` 速度环, `0x03` 位置环)
- `DATA[3] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

## 8. 获取版本号反馈
### 8.1 版本号查询指令（下发查询）
- `DATA[0]`: ID
- `DATA[1]`: 0xFD
- `DATA[2] ~ DATA[8]`: 0
- `DATA[9]`: CRC8

### 8.2 版本号反馈回复（读取）
- `DATA[0]`: ID
- `DATA[1]`: 0xFD
- `DATA[2]`: 年
- `DATA[3]`: 月
- `DATA[4]`: 日
- `DATA[5]`: 电机型号
- `DATA[6]`: 软件版本
- `DATA[7]`: 硬件版本
- `DATA[8]`: 保留
- `DATA[9]`: CRC8

    年/月/日：默认省略 20XX 的 20，例如：2021 为 `0x15`，11 月为 `0x0B`，28 号为 `0x1C`。
